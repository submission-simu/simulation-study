# simulation-study
Simulation study for 'Distributed Semiparametric Inference for Heterogeneous Sites via Single-Round Communication''

Run "main_simu.R" to reproduce Table 1 in the manuscript.

Please modify the number of cores for parallel in Line 659 of main_simu.R.

To reproduce Tables 3 and 4 in the supplement, please change the value of rho in Line 17 of main_simu.R.
